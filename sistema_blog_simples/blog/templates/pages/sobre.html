Escopo do Projeto Web em Django

Objetivo:

Desenvolver um projeto web utilizando o framework Django, proporcionando aos alunos uma introdução prática aos conceitos fundamentais de programação web e ao desenvolvimento de aplicações web dinâmicas.

Tecnologias:

-Django: Framework Python para desenvolvimento web.
-Python: Linguagem de programação.
-HTML: Linguagem de marcação de hipertexto.
-CSS: Linguagem de estilo para definir a aparência das páginas web.
-Bootstrap: Framework CSS para design responsivo e desenvolvimento front-end.
-Banco de dados: SQLite (por padrão no Django), mas pode ser substituído por outros como PostgreSQL ou MySQL.

Conceitos a serem abordados:

-CRUD: Criação, Leitura, Atualização e Deleção de dados.
-Modelos: Representação das estruturas de dados do aplicativo.
-Views: Lógica de processamento das requisições HTTP.
-Templates: Criação das interfaces visuais das páginas.
-Context: Passagem de dados dos views para os templates.
-Sessões: Manutenção de estado entre as requisições.
-Herança de templates: Reutilização de blocos de HTML em diferentes templates.
-URLs: Mapeamento de URLs para as views.
-Forms: Criação de formulários para entrada de dados.

Etapas do projeto:

-Previsão de Entregas Aula 01/07/2026
   Configuração do ambiente:
   Instalação do Python e do Django.
   Criação do projeto Django.
   Configuração do banco de dados.
   Criação dos modelos:
   Definição dos modelos para representar os dados do aplicativo (ex: usuários, posts, comentários).
   Criação das migrations e aplicação das mesmas no banco de dados.
-Previsão de Entregas Aula 03/07/2026
   Criação das views:
   Criação dos templates: Criação dos templates utilizando HTML e CSS.
   Utilização de Bootstrap para estilização.
   Utilização de herança de templates para organizar a estrutura dos templates.
   Configuração das URLs:
   Mapeamento das URLs para as views.
   Autenticação de usuários.
-Previsão de Entregas Aula 06/07/2026
   Ajsutes Finais no projeto
   Apresentação
   Entrega Final e encerramento.

Exemplo de aplicação:

BLOG SIMPLES, onde os usuários podem:
Cadastro de usuários.
Login e logout.
Criação de posts.
Edição e exclusão apenas dos próprios posts.
Visualização da lista de posts.
Visualização dos detalhes de um post.
Cadastro de comentários em um post.
Busca de posts por título ou conteúdo.
